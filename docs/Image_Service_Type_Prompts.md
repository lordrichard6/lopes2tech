# Image Service Type Prompts

## Style Guide
**Style:** Detailed technical infographic illustration, isometric view, knolling style, clean flat vector art, bold outlines, schematic annotations.
**Background:** Dark Slate Blue (matches website service card theme).
**Format:** Instagram Story (9:16 Aspect Ratio).
